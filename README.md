# Starlink Fix

This repo contains programs and documentation related to cruiser use of
Starlink's location data in conjunction with GPS data to establish
a boat's fix in GPS-compromised environments.

## Documentation

Most of the documentation for how to configure starlink and signalk and to
setup and use the repo contents is on the
[github pages website for this project](https://https://Maddox-zephyr.github.io/starlink_fix)
