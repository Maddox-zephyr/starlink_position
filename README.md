# Starlink Position

This repo contains programs and documentation related to obtaining a
position fix from Starlink, and cruiser use of
Starlink's position fix data in conjunction with GPS data to establish
a boat's fix in GPS-compromised environments.

## Documentation

Most of the documentation for how to configure starlink, OpenCPN
and signalk and to setup and use the repo contents is on the
[github pages website for this project](https://Maddox-zephyr.github.io/starlink_position)
